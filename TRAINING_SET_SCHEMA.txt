## IMPORTANT NOTES:

1. **Favorite/Underdog Perspective**: All stats are organized by betting favorite vs underdog, NOT home/away
   - Must determine favorite from betting odds
   - Then map team stats to Fav/Dog columns

2. **Calculated Fields**: Fields with "/" (e.g., "runs/AB") are ratios calculated from raw values

3. **Bullpen Stats**: BP columns are team pitching stats MINUS starting pitcher contribution

4. **Critical Gap**: Columns 20-21 (Fav SP/Dog SP) require starting pitcher names
   - Balldontlie API doesn't have direct "probable pitchers" endpoint
   - Need alternative solution

5. **Missing from API**:
   - Pitching doubles/triples (cols 56-63)
   - Pitches per inning (cols 102-103)
   - Games finished (cols 104-105)

6. **Name Mapping**:
   - CHW (API) → CWS (training set)
   - KC (API) → KAN (training set)
